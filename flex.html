---------------------------FLEXBOX-------------------------------------

--------------СВОЙСТВА ДЛЯ ФЛЕКС-КОНТЕЙНЕРА (ЖЕЛТЫЙ)---------------

display: flex; включает flex разметку, - это значение по умолчанию
1.Все элементы выстраиваются в ряд
2.Элемент - блочный (занимает всю ширину области)
3.При уменьшении экрана, элементы не перемещаются на другую строку
display: inline-flex;
1.Элемент - строчно-блочный (занимает столько места, сколько в нем контента)
2.Если родителю этого контента (.block) задать text-align:centr; то весь контейнер встанет в центр

---------------------ПО ОСНОВНОЙ ОСИ---------------

justify-content: flex-start; - это значение по умолчанию. Все флекс элементы располагаются в начале (слева) контейнера
justify-content: flex-en; Все флекс элементы располагаются в конце (справа) контейнера
justify-content: center; Все флекс элементы располагаются в центре контейнера
justify-content: space-between; Все флекс элементы располагаются на одном расстоянии друг от друга
justify-content: space-around; Все флекс элементы имеют одинаковые отступы с двух сторон

---------------------ПО ВТОРОСТЕПЕННОЙ ОСИ---------------------

align-items: stretch; - это значение по умолчанию. Все флекс элементы растягиваются под самый высокий элемент
align-items: flex-start; Все флекс элементы прижимаются к верху контейнера
align-items: flex-end; Все флекс элементы прижимаются к низу контейнера
align-items: center; Все флекс элементы распологаются по-центру
align-items: baseline; Все флекс элементы выстраиваются по базовой линии
--------------------------------------------------------
flex-wrap: nowrap; - это значение по умолчанию. Регулирует, чтобы флекс-элементы не перескакивали на другую строку
flex-wrap: wrap; При уменьшении места, элементы переходят на другую строку
flex-wrap: wrap-reverse; При уменьшении места, элементы переходят на другую строку, но наоборот (3,2,1)
------------------------------------------------------------------------
flex-direction: row; Устанавливает основную ось, по которой выстраиваются флекс-элементы. Значение по-умолчанию "row" (ряд)
flex-direction: row-reverse; Разворачивает ряд элементов в обратном порядке
flex-direction: column; Выстраивет элементы в колонку
flex-direction: column-reverse; Выстраивает элементы в обратнем порядке

--------------СВОЙСТВА ДЛЯ ФЛЕКС-ЭЛЕМЕНТОВ (ФИОЛЕТОВЫЙ)---------------

align-self: stretch; То-же самое, что и align-items для контейнера.
1.Перезаписывает значения для флекс-элементов (например для контейнера стоит значение align-items: center; но для отдельного элемента стоит align-self: stretch; значит для этого элемента будет действовать свое значение "stretch")
2.Все значения такие-же как и для align-items: stretch; flex-start;flex-end; center; baseline;
3.Для каждого флекс-элемента можно задавать свое значение "align-self"
------------------------------------------------------------------------
order - Выводит элементы в том порядке, в каком мы укажем:
.block__column_1 {
order:3;
}
.block__column_2 {
order:2;
}
.block__column_3 {
order:1;
}
------------------------------------------------------------------------
flex-basis: auto; Отвечает за базовый размер элемента, но это не то-же самое, что и width.
flex-basis: 500px; указывается в px и в %
flex-basis: 30%;
flex-grow: 0; Отвечает за увеличение относительно flex-basis:, по-умолчанию увеличение запрещено "0" Если выставить "1", то элементы увеличатся.
flex-shrink: 1; Отвечает за уменьшение относительно flex-basis:, по-умолчанию уменьшение разрешено "1" Если выставить "0", то элементы перестанут уменьшатся.

flex: 0 1 auto; Это короткая запись flex: grow shrink basis;
